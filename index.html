Sugeng Rawuh!

Aksara Jawa atau Hanacaraka adalah sistem tulisan tradisional yang digunakan untuk menulis bahasa Jawa. Mari kita pelajari bersama mulai dari aksara dasar, pasangan, sandhangan, hingga latihan menulis dan membaca aksara Jawa.

Tips Belajar:

  • Mulai dari aksara dasar dulu sampai hafal
  • Pahami sistem pasangan untuk menulis konsonan rangkap
  • Pelajari sandhangan untuk vokal dan konsonan akhir
  • Latihan terus menerus dengan soal-soal
Progress Belajar: Aksara Dasar

ꦧꦒ

Progress Belajar: Pasangan

Pasangan Aksara

Pasangan digunakan untuk menuliskan konsonan rangkap atau konsonan yang tidak diikuti vokal. Pasangan ditulis di bawah aksara sebelumnya.

Aksara Bunyi Pasangan Contoh Kata Arti
ꦲ ha ꧀ꦲ ꦱꦃꦲ (saha) siapa
ꦤ na ꧀ꦤ ꦱꦸꦤ꧀ꦤ (sunna) sunah
ꦕ ca ꧀ꦕ ꦏꦕ꧀ꦕ (kacca) kaca
ꦫ ra ꧀ꦫ ꦒꦼꦣ꧀ꦫ (gedra) cepat
ꦏ ka ꧀ꦏ ꦧꦏ꧀ꦏ (bakka) sudah
ꦢ da ꧀ꦢ ꦧꦢ꧀ꦢ (badda) bandel
ꦠ ta ꧀ꦠ ꦠꦠ꧀ꦠ (tatta) tata
ꦱ sa ꧀ꦱ ꦏꦱ꧀ꦱ (kassa) kasur

Catatan Penting:

Pasangan selalu ditulis MENEMPEL di bawah aksara yang mendahuluinya. Fungsinya untuk mematikan vokal 'a' pada aksara sebelumnya dan menambahkan bunyi konsonan baru.

Progress Belajar: Sandhangan

Sandhangan (Tanda Baca)

Sandhangan adalah tanda diakritik yang digunakan untuk mengubah bunyi vokal atau menambahkan konsonan akhir pada aksara Jawa.

Sandhangan Swara (Vokal):

Wulu ( ꦶ )

Mengubah vokal 'a' menjadi 'i'

Contoh: ꦏꦶ (ki), ꦱꦶꦏ (sik)

Suku ( ꦸ )

Mengubah vokal 'a' menjadi 'u'

Contoh: ꦏꦸ (ku), ꦒꦸꦫꦸ (guru)

Pepet ( ꦼ )

Mengubah vokal 'a' menjadi 'è/e'

Contoh: ꦏꦼ (ke), ꦱꦼꦒ (seg)

Taling ( ꦺ )

Mengubah vokal 'a' menjadi 'é'

Contoh: ꦏꦺ (ké), ꦱꦺꦒ (ség)

Taling Tarung ( ꦺꦴ )

Mengubah vokal 'a' menjadi 'o'

Contoh: ꦏꦺꦴ (ko), ꦠꦺꦴꦏ (tok)

Sandhangan Panyigeg (Konsonan Akhir):

Wignyan ( ꦃ )

Konsonan akhir 'h'

Contoh: ꦲꦺꦴꦃ (oh), ꦱꦃ (sah)

Layar ( ꦂ )

Konsonan akhir 'r'

Contoh: ꦲꦺꦴꦂ (or), ꦏꦂ (kar)

Cecak ( ꦁ )

Konsonan akhir 'ng'

Contoh: ꦲꦺꦴꦁ (ong), ꦒꦼꦁ (geng)

Pengkal ( ꦀ )

Konsonan akhir 'ng' (digunakan untuk kata benda)

Contoh: ꦲꦀ (ang), ꦒꦀ (gang)

Contoh Lengkap dengan Sandhangan:

ꦧꦸꦏꦸ
buku (buku)
ꦒꦸꦫꦸ
guru (guru)
ꦱꦶꦏꦶꦭ꧀
sikil (kaki)
ꦲꦺꦴꦩꦃ
omah (rumah)
ꦏꦺꦴꦤ꧀ꦕꦺꦴ
konco (teman)
